# Design Text by Woxro

A visually engaging project built with Next.js App Router that features dynamic, animated text design by Woxro. This project leverages smooth scrolling with Lenis and advanced animations with GSAP to create a modern and immersive user experience.

## Table of Contents

- [Overview](#overview)
- [Features](#features)

## Overview

Design Text by Woxro is a Next.js project that combines the power of the new App Router with smooth scrolling and captivating animations. The project demonstrates how to integrate Lenis for a fluid scrolling experience and GSAP for robust animations, resulting in an interactive design text presentation.

## Features

- **Modern Routing:** Built with Next.js using the new App Router for improved performance and routing flexibility.
- **Smooth Scrolling:** Integrated Lenis provides a smooth scrolling experience.
- **Advanced Animations:** GSAP is used to animate text transitions, fades, and transformations.
- **Responsive Design:** The layout is fully responsive across devices.
- **Optimized Performance:** Follows Next.js best practices for fast, efficient rendering.

## Technologies Used

- **Next.js 13+ (App Router)**
- **React**
- **GSAP (GreenSock Animation Platform)**
- **Lenis (Smooth scrolling library)**
- **Tailwind CSS** (or alternative styling method)
- **Vercel** (for deployment)
